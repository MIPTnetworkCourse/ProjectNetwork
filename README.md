# ProjectNetwork
Классификация текстов(новостей) по категориям с помощью библиотеки Keras. Модель обучается на данных, загруженных с новостного ресурса. Веб-интерфейс реализован на Flask.

Grab.py - загрузка новостей с сайта fontanka.ru

Learn.py - обучение модели

Classify.py - веб-интерфейс классификатора
